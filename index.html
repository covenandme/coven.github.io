Hexo + Github pages 搭建个人博客
发表于2026-06-18|日常
用 Hexo + GitHub Pages 简单搭了一个个人主页。 都是开国源码了Hexo 装起来倒是简单，Node.js 跑一下就行： 1234npm install -g hexo-clihexo init my-blogcd my-blognpm install 然后选了个叫 Butterfly 的主题： 1npm install hexo-theme-butterfly 再配一下 _config.yml 里把主题切过去，又搞了个 _config.butterfly.yml 自己配置。 部署到 GitHub Pages这一步很简单，装个部署插件，配一下仓库地址就好了： 1npm install hexo-deployer-git --save 1234deploy: type: git repo: git@github.com:covenandme/coven.github.io.git branch: main 写完文章，用 hexo deploy 就发布上去了。 顺手调了调样子 但是感觉调不明白，如何做到简约但高端大气上档次??
